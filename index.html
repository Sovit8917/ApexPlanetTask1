HTML: Building Blocks of the Web

This is a paragraph about HTML, which stands for HyperText Markup Language. It is used to structure content on the web.

  • Headings
  • Paragraphs
  • Images
  • Links

CSS: Styling the Web

CSS (Cascading Style Sheets) is used to style HTML elements. You can change colors, fonts, spacing, and layout.

  • Font styling
  • Color and background
  • Margins and padding
  • Responsive design

JavaScript: Making Webpages Interactive

JavaScript is used to create dynamic interactions on webpages. For example, you can make buttons respond to clicks.
